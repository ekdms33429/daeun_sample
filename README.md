## Test Automation Using Appium and Python (iOS)

#### 환경 구성
- Appium 2.0.0
- Python3
- PyCharm
- Appium Inspector
- Node.js
- Java JDK 8

#### Device Info
* iPhone 14 Pro(Real Device)

#### Test Scenario
> Nike 앱 실행 > 로그인 > 웹뷰 이동 > 이메일, 약관 동의, 비밀번호 입력 > 로그인 성공

### 주요 코드
- Base.action_func: UI 동작 관련
- Base.check_func: 유효성 검사 관련
- Pages.LoginPage: 로그인 기능 관련
